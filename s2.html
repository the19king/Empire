تعليمات تيرمكس

apt update

يقوم هذا الأمر بتحديث قائمة الحزم المتوفرة من المصادر. يُنصح باستخدامه قبل تثبيت أو ترقية أي حزمة.

apt update
apt upgrade

يقوم بترقية الحزم المثبتة إلى أحدث إصدار متاح في المصادر. بعد تحديث القائمة باستخدام apt update، استخدم هذا الأمر لترقية النظام.

apt upgrade
pkg install [اسم الحزمة]

يستخدم لتثبيت الحزم. على سبيل المثال، لتثبيت Python:

pkg install python
termux-setup-storage

يُستخدم هذا الأمر للسماح لتطبيق تيرمكس بالوصول إلى ذاكرة الجهاز (التخزين الداخلي والخارجي).

termux-setup-storage
ls

يعرض قائمة الملفات والمجلدات في الدليل الحالي. يمكن استخدام خيارات مثل -la لتعزيز المعلومات.

ls -la
cd

يستخدم لتغيير الدليل الحالي. على سبيل المثال، للدخول إلى مجلد التخزين الخاص بالتنزيلات:

cd ~/storage/downloads
mkdir

يستخدم لإنشاء مجلد جديد. على سبيل المثال، لإنشاء مجلد باسم "MyFolder":

mkdir MyFolder
