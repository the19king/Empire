أشهر أدوات اختبار الاختراق بلغة بايثون

Scapy

مكتبة قوية لمعالجة حزم الشبكة ويمكن استخدامها لإنشاء حزم مخصصة وتحليل حركة المرور.

pip install scapy

Metasploit Framework

إطار عمل شهير لاختبار الاختراق مع واجهة بايثون تسمى MSFRPC.

pip install pymetasploit3

Impacket

مجموعة من الفئات لمعالجة بروتوكولات الشبكة بمستوى منخفض.

pip install impacket

Requests

مكتبة شهيرة لتنفيذ طلبات HTTP والتي يمكن استخدامها في اختبار تطبيقات الويب.

pip install requests
